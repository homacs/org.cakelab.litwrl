Life in the Woods Renaissance

Manual Installation

last updated: ${system.date}


Downloads:

  • Life in the Woods: Renaissance. Chose one of the following packages:
    • Basic
    • Hungry
  • Forge-1.7.10-10.13.2.1230

Installation

Minecraft 1.7.10

  • Create a new profile in the Minecraft launcher
  • Select the Minecraft version 1.7.10
  • Hit play to install this version
  • Close Minecraft and the Minecraft launcher.

Forge

  1. Run the downloaded jar forge-1.7.10-10.13.2.1230-installer.jar (e.g. java -jar forge-1.7.10-10.13.2.1230-installer.jar or double click on it)
  2. Click on OK
  3. Done.

Life in the Woods: Renaissance

  1. Modify your new profile to run with Forge
    1. Start the Minecraft launcher again
    2. In the left bottom corner, select the profile you have created before.
    3. Then click on New Profile
    4. Enter a suitable name for the profile (e.g. LitwR.Basic)
    5. Click on Game Directory and chose a folder name (e.g. <yourhomedirectory>/MinecraftGames/LitWR.Basic)
    6. Remember the location of the game directory, because you will need it later.
    7. In the Section Version Selection check what it says under Use version: and if it does not show release 1.7.10-Forge10.13.2.1230 then select it in the drop down list.
    8. Click on Save Profile
    9. Start Minecraft once. Do not create a world yet, just close Minecraft when you see its main menu.
    10. Close the Minecraft launcher.
  2. Locate the game directory you have created with the profile (see above)
  3. Unzip the Life in the Woods Renaissance package you have downloaded inside the game directory (e.g. inside <yourhomedirectory>/MinecraftGames/LitWR.Basic).
  4. Start the Minecraft launcher again.
  5. Select the profile you have created
  6. Start the game.
  7. Have fun!

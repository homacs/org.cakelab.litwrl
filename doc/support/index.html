Solving Issues

If you have encountered any issue with Life in the Woods Renaissance or any components involved please consider the following steps to solve it.

  • Make sure, you have followed the installation instructions which also come with the downloaded zip file.
  • FAQ page. The FAQ page contains a list of formerly asked questions and solutions to solve issues.
  • Filed Bugs and Feature Requests. If you haven't found a solution in the FAQs then search the list of filed bugs and feature requests.
  • Send a Bug or Feature Request. If your specific issue or request has not been filed yet, then go ahead and tell us so we can try to find a solution.
